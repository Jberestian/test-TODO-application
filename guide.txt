пошаговое руководство по созданию апликухи в Императивном стиле

1.разметка

    1.main

    2.header

    3.ul
        li
            input.checkbox type="checkbox"
            label.title
            input.textfield type="text"
            button.edit
            button.delete

    4.form id="todo-form"
        input type="text"
        button type="submit"


2.стилизация


3.js

    1. создаем переменные для доступа
    2. привязываем обработчик события на отправку формы
    3. создаем функцию-обработчик для создания таска
        function addTodoItem(event)
        1. остановка отправки данных на сервер.Иначе страница  перезагрузится
        2. проверка на значение в addInput
        3. пишем функцию для создания таска
        4. обнуляем значение add-input в todo-form
    4. пишем функцию создания таска
        1. создаем елемент checkbox
            1. добавляем чекбоксу тип
            2. добавляем чекбоксу класс
        2. создаем  елемент label
            2.1. добавляем ему текст(параметр,который принимает функция)
            2.2. добавляем ему класс
        3. создаем елемент инпут текст
            3.1. добавляем ему тип
            3.2. добавляем ему класс
        4. создаем елемент button для изменения таска
            4.1 добавляем ему текст
            4.2 добавляем ему класс
        5. создаем елемент button для удаления текста таска
            5.1 добавляем ему текст
            5.2 добавляем ему класс
        6. создаем сам таск (основной елемент)
            6.1 добавляем ему класс
            6.2 добавляем таску елемент checkbox
            6.3 добавляем таску елемент label
            6.4 добавляем таску елемент editInput
            6.5 добавляем таску елемент editButton
            6.6 добавляем таску елемент deleteButton
            6.7 возвращаем созданный таск
    5. добавляем таск в список todo-list
    6. обнуляем значение add-input в todo-form
    7. пишем функцию для привязки событий при создании таска bindEvents
        7.1 получаем доступ к checkbox
        7.2 получаем доступ к кнопке editButton
        7.3 получаем доступ к кнопке deleteButton
        7.4 подписываемся на события у елементов таска
    8. пишем функцию для работы с checkbox
        8.1 получаем доступ к таску (li) через родителя элемента
        8.2 меняем визуально состояние таска при активации checkbox
    9. пишем функцию 'изменить' таска
        9.1 получаем доступ к таску (li) через родителя элемента
        9.2 получаем доступ к названию таска
        9.3 получаем доступ к полю для редактирования таска
        9.4 получаем доступ на проверку наличия свойства у таска
        9.5 проверяем наличие свойства
        9.6 присваеваем новое название таска
        9.7 получаем доступ к кнопке editButton
        9.8 если не находимся в режиме редактирование ,то заходим в него
        9.9 делаем обратное 9.6
        9.10 убираем класс 'editing'
    10. пишем функцию удаления таска
        10.1 получаем доступ к таску (li) через родителя элемента
        10.2 удаляем таск
    11. пишем основную функцию main
        11.1 делаем подписку на форму
        11.2 добавляем обработчики на елементы таска
